Projects

Real Time Voice Translator


The Real-Time Voice Translator is a software application that enables seamless communication across different languages. It converts spoken language into text, translates it into the target language, and then uses text-to-speech to communicate the translation in real-time. This project integrates speech recognition, machine translation, and text-to-speech synthesis to bridge language barriers, making it ideal for travel, business, and education.

Technologies Used: Python, Google Cloud Speech-to-Text, Google Cloud Translation API, Text-to-Speech..
